# 🔄 Transition Guide: IQ Bot → Deriv Bot (v5.0.0)

This guide explains the key differences and how to adapt your workflow from the IQ Option Bot (`iq_bot`) to the new Deriv Bot (`deriv_bot`).

## 🚨 Key Differences

| Feature | IQ Bot (`iq_bot`) | Deriv Bot (`deriv_bot`) |
| :--- | :--- | :--- |
| **Connection** | HTTP Polling (Check every 1-3s) | **Real-time Streaming** (WebSocket v4.0+) |
| **Assets** | Forex, Stocks, Crypto (OTC) | **Volatility Indices** (Synthetic 24/7) |
| **Market Hours** | Mon-Fri (OTC on weekends) | **24/7 Non-Stop** (365 days/year) |
| **Strategy** | Price Action, Patterns, Indicators | **Adaptive Volatility** (TREND vs PULLBACK) |
| **Config** | JSON/INI | **Asset Profiles** (`asset_profiles.json`) |
| **Payouts** | 70% - 95% (Variable) | **~95%** (Consistent for Rise/Fall) |

## 🛠️ Configuration Changes (`config.py` & `asset_profiles.json`)

### 1. Asset Names
*   **IQ Bot:** `EURUSD`, `EURUSD-OTC`
*   **Deriv Bot:**
    *   `1HZ100V`: Volatility 100 (1s) Index (Fastest)
    *   `R_75`: Volatility 75 Index (Trending)
    *   `1HZ10V`: Volatility 10 (1s) Index (Low Volatility)

### 2. Money Management
Deriv's minimum trade amount is often around **$0.35 - $0.50** (depending on asset).
*   **Recommendation:** Use `asset_profiles.json` to set specific amounts per asset if needed.

### 3. API Keys
*   **Deriv Bot:** **API Token** (Create at [Deriv Settings](https://app.deriv.com/account/api-token)).
    *   Scopes required: `read`, `trade`, `trading_information`.

## 💡 Strategy Tips for Deriv

Volatility Indices are **synthetic markets** generated by algorithms:
1.  **Adaptive Logic (V5.0):** The bot now detects market regimes (`HIGH_VOL`, `NORMAL`, `LOW_VOL`) and switches strategies automatically.
2.  **No News Events:** Unlike Forex, these assets are unaffected by economic news.
3.  **High Volatility:** Prices move fast. Use the built-in **Tick Velocity Guard** and **Regime Detection** for safety.

## 🚀 Migration Checklist

1.  [ ] Get Deriv API Token from website.
2.  [ ] Put Token in `.env` file (`DERIV_API_TOKEN`).
3.  [ ] Run `test_deriv_connection.py` to confirm login.
4.  [ ] Run `scripts/backtest_7d.py` to verify strategy on your favorite asset.
5.  [ ] Start `bot.py` and monitor via the Dashboard (Port 5001).
